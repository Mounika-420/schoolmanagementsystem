# School Management System

## 📌 Project Overview

This is a **simple School Management System web application** built using **HTML, CSS, and JavaScript**. It allows users to:

- Add new students with their **Name, Grade, and Age**.
- View a list of added students in a table format.
- Delete students from the list dynamically.

---

## 🚀 Features

✅ Clean and responsive UI  
✅ Add student details dynamically  
✅ Delete student entries easily  
✅ Built entirely with vanilla JavaScript (no external frameworks)

---

## 💻 Files Included

- **index.html** – main HTML structure containing embedded CSS and JavaScript  
- *(No external CSS or JS files used; all code is in one HTML file)*

## 🔧 How to Run

1. **Download or clone this repository.**

2. Navigate to the project folder.

3. **Open `index.html`** directly in your web browser (Chrome, Firefox, Edge).

4. You will see the School Management System page where you can:

   - Enter student name, grade, and age in the form.
   - Click **“Add Student”** to add them to the table.
   - Click **“Delete”** to remove a student from the table.

---

## 📝 Code Explanation

### HTML

- A form with inputs for **Name, Grade, and Age** and a submit button.
- A table displaying the list of students with their details and a delete action.

### CSS

- Embedded within `<style>` tags.
- Provides layout styling, form design, table formatting, and button hover effects.

### JavaScript

- Handles **form submission**, dynamically creates table rows, and appends them to the table body.
- Includes a `deleteStudent()` function to remove specific student entries when the Delete button is clicked.
